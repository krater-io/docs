Currently, the Krater Node installation process for Windows involves the following steps:

1. Once you have successfully created a Location, you can proceed to download the Krater Node installer file.
<Frame caption=""><img height="200" src="https://github.com/krater-io/docs/blob/main/images/krater-node/install-windows/download.webp?raw=true" /></Frame>

2. Navigate to the Start Menu, search for 'Powershell', right-click on it and choose 'Run as Administrator'.
<Frame caption=""><img height="200" src="https://github.com/krater-io/docs/blob/main/images/krater-node/install-windows/powershell.webp?raw=true" /></Frame>

3. Inside Powershell, use the "cd [YourPath]" command to navigate to the location of your downloaded installer file. Replace [YourPath] with the actual path where your installer is located.
<Frame caption=""><img height="200" src="https://github.com/krater-io/docs/blob/main/images/krater-node/install-windows/cd-directory.webp?raw=true" /></Frame>

4. Type in the 'Set-ExecutionPolicy unrestricted' command to permit the script to run, then press enter.
<Frame caption=""><img height="200" src="https://github.com/krater-io/docs/blob/main/images/krater-node/install-windows/change-policy.webp?raw=true" /></Frame>

5. Run the installer script by typing '.\install.ps1' and then press enter. Upon encountering a security warning, type “r” for “Run once” and press enter.
<Frame caption=""><img height="200" src="https://github.com/krater-io/docs/blob/main/images/krater-node/install-windows/execute-installer.webp?raw=true" /></Frame>

6. A disclaimer and terms of service may appear. If you agree, type “y” to accept and proceed.
<Frame caption=""><img height="200" src="https://github.com/krater-io/docs/blob/main/images/krater-node/install-windows/agree-to-terms.webp?raw=true" /></Frame>

7. When prompted, select a directory to install Krater Node. Importantly, the chosen folder should NOT be inside the “Users” directory. For instance, avoid places like C:\Users\Desktop or other subfolders within “Users”.
<Frame caption=""><img height="200" src="https://github.com/krater-io/docs/blob/main/images/krater-node/install-windows/select-folder.webp?raw=true" /></Frame>

8. Lastly, the system will request your Krater Node ID. Input this ID from your Krater Node dashboard and press enter to link the installed software with the backend.
<Frame caption=""><img height="200" src="https://github.com/krater-io/docs/blob/main/images/krater-node/install-windows/input-id.webp?raw=true" /></Frame>

Once you've successfully completed these steps, the Krater Node should be up and running on your Windows machine. It's designed to automatically restart, even if your server goes down and comes back up, thus ensuring continuity.

Note: Given the way Windows handles scripts, if you encounter an error during or after the installation, don't hesitate to reach out to us for assistance.
